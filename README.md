# Learn Core Java
 
A structured and beginner-friendly repository to learn and practice Core Java concepts. This project is organized by topics, covering everything from fundamentals to advanced concepts like Collections, Multi-threading, and Java 8 features.
 
## 🔗 Clone the Repository
 
```bash
git clone https://github.com/rahamath-codes/Learn-CoreJava.git
```

## 📚 Topics Covered
 
```plaintext
1. Fundamentals
   - Data Types
   - Operators
   - Keywords
   - Hello World Program
 
2. Access Modifiers
   - public, private, protected, default
 
3. Class, Methods, and Objects
   - Object Creation
   - Instance vs Static
   - Method Overloading
 
4. Control Statements
   - if, if-else, nested if
   - switch
 
5. Looping Statements
   - for loop
   - while loop
   - do-while loop
 
6. Abstraction
   - Abstract Classes
   - Interfaces
 
7. Encapsulation
   - Private variables
   - Getters & Setters
 
8. Exception Handling
   - try-catch-finally
   - throw & throws
   - Checked and Unchecked exceptions
 
9. Inheritance
   - Single, Multilevel, Hierarchical
   - super keyword
 
10. Polymorphism
    - Compile-time (Method Overloading)
    - Run-time (Method Overriding)
 
11. Scanner Class
    - java.util.Scanner for user input
 
12. String Handling
    - String methods
    - StringBuilder vs StringBuffer
 
13. Typecasting
    - Widening and Narrowing
 
14. Variable Scope
    - Local, Instance, and Static variables
 
15. Java Arrays
    - Single and Multidimensional arrays
 
16. Java Collections
    - List, Set, Map
    - ArrayList, LinkedList
    - HashSet, TreeSet
    - HashMap, TreeMap
    - Comparable and Comparator examples
 
17. Java Multi-threading
    - Thread class and Runnable interface
    - Thread lifecycle
    - Synchronization
 
18. Java 8 Features
    - Lambda Expressions
    - Functional Interfaces
    - Stream API basics
    - Method References
 
19. Packages
    - User-defined packages
    - Import statements
